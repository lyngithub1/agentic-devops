Agentic DevOps — Product Idea Pipeline

A Vite + React + TypeScript demo that visualizes a Magentic, multi-agent "idea-to-market" pipeline. It runs in two modes: a deterministic Simulation from curated content, and a Live mode where Azure OpenAI authors each phase in real time.

Data flow

How a run travels through the system. Use the toggle above to switch paths.

  1. User picks a business and presses Run in Simulation mode.
  2. store.start() assembles a RunScript via buildSegments() from curated content.
  3. SimulatedRunner plays steps at the chosen speed, emitting phase / sub-agent / artifact / activity events.
  4. Events update the Zustand store.
  5. React re-renders the lanes, artifacts, orchestrator narration and gates.
  6. Human-approval gates pause playback until the reviewer approves.
  1. On load, checkLiveHealth() GETs /api/llm/health → the model registry (fast / deep).
  2. User selects a model (GPT-4.1 or GPT-5.4) and presses Run in Live mode.
  3. _startLive builds a baseline, then for each phase calls applyLiveStage().
  4. llmComplete() POSTs the system + stage prompt (and model id) to /api/llm.
  5. The dev proxy resolves that model's env, fetches an Entra token (cached per subscription) and calls Azure OpenAI.
  6. The model returns JSON → coerce.ts validates & shapes it into typed artifacts.
  7. The phase's segment is rebuilt and played by SimulatedRunner — identical animation to Simulation.
  8. Store updates → UI renders real, company-specific content, phase by phase.
